VBO (Vertex Buffer Object) - это буффер с данными о вершинах/нормалями/цветами, загружаемый в видеокарту
VAO (Vertex Array Object) - это объект связей данных VBO и VertexAttribute. То-есть он хранит привязку значений конкретного буфера с входным атрибутом (помеченный location) в шейдере
Квалификатор layout (переменная-атрибут) (в вершинном шейдере) - это указатель расположения атрибута вершин
Квалификатор uniform - отлично подходят для работы с матричными значениями, доступны только для чтения. Записывать данные в нее можно только из основной программы OpenGL

* Атрибуты вершин – это входные переменные для вершинного шейдера
* VAO содержит информацию о связях между данными в буферах и входными атрибутами вершин
* С помощью квалификатора layout каждому атрибуту в вершинном шейдере 
назначается свой индекс.
* Для каждого атрибута создается и заполняется данными свой буферный 
объект.
* Пока соответствующий буфер связан с целевой точкой привязки, вызовом 
glVertexAttribPointer создается и определяется объект массива вершин.
* Перед отображением выполняется связывание объекта массива вершин и 
вызывается функция glDrawArrays или другая подобная функция рисования 
(например, glDrawElements)
